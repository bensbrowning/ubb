Usage:

```
mv ~/.bashrc ~/.bashrc.pre-ubb 
mv ~/.bash_profile .bash_profile.pre-ubb
ln -s /path/to/ubb/git/repo/bashrc ~/.bashrc
ln -s /path/to/ubb/git/repo/bash_profile ~/.bash_profile
mkdir ~/.ubb_config
```
